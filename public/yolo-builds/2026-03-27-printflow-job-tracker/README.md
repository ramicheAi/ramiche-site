# PrintFlow Job Tracker

Interactive 3D print production management for Ramiche Studio.

## What It Does

Full job lifecycle tracker — from client intake through printing to delivery. Manages the Bambu Lab A1 production queue with real-time cost quoting, status progression, and per-job timelines.

## Features

- **Job Queue Management** — Create, queue, track, and complete print jobs
- **Live Cost Quoting** — 6-material database (PLA/PLA+/PETG/ASA/ABS/TPU) with weight, time, labor, depreciation, and 40% margin
- **Status Pipeline** — Review → Queued → Printing → Done (with failure/requeue)
- **Rush Pricing** — 25% surcharge on rush orders, auto-sorted to top
- **Job Detail Expansion** — Temp settings, layer height, timeline history per job
- **Progress Tracking** — Live progress bars for active prints
- **Revenue Dashboard** — Running total of completed job revenue
- **localStorage Persistence** — Jobs survive page refresh
