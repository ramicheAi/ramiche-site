# METTLE // Practice Builder

Structured practice design tool for competitive swim coaches. Build complete workout plans with drag-and-drop set ordering, pace calculations, and exportable practice cards.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- **7 set categories:** Warm-Up, Pre-Set, Kick, Drill, Main, Sprint, Cool-Down
- **8 strokes/modalities** with real pace data across 4 skill levels
- **5 intensity zones** (Easy → Max Effort) with time estimation
- **Drag-and-drop** set reordering
- **7 quick templates** including a full 4000-yard practice
- **Live summary bar:** total yardage, set count, estimated time, avg intensity, primary stroke
- **Export:** text practice card (clipboard or print)
- **Pace reference panel:** click any stroke to auto-fill interval

## What's Missing / Next Steps

- Save/load practices to localStorage
- Multi-group support (e.g., Group A sprint while Group B distance)
- Equipment annotations (paddles, fins, snorkel)
- Integration with METTLE athlete profiles for personalized pacing
- PDF export with METTLE branding
