# METTLE // Team Strength Heat Map

Visual team depth analysis across all 12 standard swim events. Identifies event deserts, scoring potential, and recruiting priorities.

## How to Run
Open `index.html` in any browser. No dependencies.

## Features
- **Roster Management:** Add swimmers with times per event, or load a 14-swimmer sample team
- **Heat Map:** Color-coded matrix (AA/A/BB/B/Novice) showing every swimmer × every event. Depth row shows entry count. Tooltip reveals time + standard
- **Scoring Potential:** Simulates dual meet points (6-4-2 scoring), full depth chart per event with ranked swimmers
- **Gap Analysis:** Identifies event deserts (0 swimmers), single-swimmer dependencies, intra-event gaps (>10% between #1 and #2), and stroke group weaknesses
- **Recruiting Recommendations:** Prioritized P1/P2/P3 recruitment targets with specific time standards needed, point impact projections, and cross-training suggestions

## What's Missing
- Age-group adjusted benchmarks (currently uses 13-14 as reference)
- CSV roster import
- Meet results integration for auto-population
- Championship meet scoring (multi-heat/prelims-finals)
