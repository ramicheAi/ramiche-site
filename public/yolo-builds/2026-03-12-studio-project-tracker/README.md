# Ramiche Studio — Project Tracker

Interactive client pipeline and project management dashboard for Ramiche Studio's creative services ($400/$1,500/$3,000/$6,000+ tiers). Track projects from inquiry through delivery with full revenue analytics.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- **5-Stage Pipeline Board:** Inquiry → Discovery → In Progress → Review → Delivered (Kanban-style)
- **Project Creation Form:** Client name, email, tier selection, custom pricing, description, source tracking, deadlines
- **Service Tier Overview:** Visual cards for Sprint/Starter/Pro/Elite with deliverables breakdown
- **Revenue Dashboard:** Total revenue, pipeline value, average project value, activity log
- **Project Detail Modal:** Notes, edit, stage advancement, deletion
- **LocalStorage Persistence:** All data persists between sessions
- **Demo Data:** 6 realistic sample projects seeded on first load
- **Ambient Particles + Heartbeat Glow:** Living Experience Philosophy (72 BPM)
