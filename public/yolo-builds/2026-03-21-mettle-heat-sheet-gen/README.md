# METTLE // Heat Sheet Generator

Interactive heat sheet generator for swim coaches — enter swimmers and seed times, choose seeding mode, generate formatted heat cards with proper lane assignments.

## Features

- **4 meet templates:** Dual (14 events), Championship (26), Sprint (8), Distance (6)
- **3 seeding modes:** Circle seed (USA Swimming standard), Straight seed, Random
- **Center-out lane assignment:** Fastest swimmers get center lanes
- **Mixed or single-gender meets** with automatic event numbering
- **Sample data loader:** 20 swimmers across 2 teams with realistic seed times
- **Print-ready:** Clean B&W print stylesheet for poolside use
- **Text export:** Downloadable .txt heat sheets

## How to Run

Open `index.html` in any browser. No build step, no dependencies.
