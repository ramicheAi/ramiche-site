# METTLE Sales Forecast Command Center

Interactive pipeline-to-revenue forecasting tool for METTLE swim coaching SaaS sales.

## What It Does

- **Deal Pipeline Manager** — Add, track, and remove deals across 6 sales stages with weighted probability
- **3-Scenario Forecast** — Commit (verbal + negotiation), Best Case (all weighted), Stretch (100% close)
- **Revenue Waterfall** — 3-month visual projection with existing MRR + new pipeline
- **Deal Velocity Metrics** — Active deals, avg ACV, win rate, avg cycle time, velocity/day and /month
- **Pipeline Coverage Ratio** — SVG ring gauge showing coverage vs quarterly target with health indicators
- **Stage Distribution** — Horizontal bar chart of pipeline value per stage
- **6-Month Projection** — Line chart with commit/best-case/stretch scenarios over time

## How to Run

Open `index.html` in any browser. No build step, no dependencies. Demo data loads automatically with 10 swim team deals.
