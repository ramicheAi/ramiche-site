# METTLE ROI & Retention Forecaster

An interactive financial modeling tool for swim club owners. It calculates the financial impact of implementing METTLE by modeling churn reduction and tier upgrade lift against the club's baseline metrics.

## How to run
Open `index.html` in any modern browser.

## Next steps
