## What It Is

A sales dashboard prototype for METTLE's sales team, focusing on three core revenue functions:

1. **Lead Scoring Matrix** — real-time scoring of swim team leads (size, engagement, stage) with color-coded badges
2. **Email Sequence Builder** — 4-step outreach sequence with performance metrics (open rates, conversion, time to close)
3. **Pipeline Projection** — stage-by-stage conversion modeling with revenue projections

## How to Run

Open `index.html` in any modern browser. No server needed.

## Features

- **Live lead scoring** with simulated updates (scores adjust every 10 seconds)
- **Pipeline stages visualization** (Prospect → Discovery → Demo → Proposal)
- **Revenue projection** based on conversion rates and current pipeline
- **Email sequence performance tracking** with real metrics
- **METTLE‑specific context**: $349/month Program tier pricing, swim team personas, retention positioning
- **Responsive design** works on desktop and tablet
