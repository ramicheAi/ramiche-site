# Agent Arbitrage Calculator

Interactive ROI calculator for the Verified Agent Business. Visualizes the cost/time arbitrage of renting a Parallax AI agent ($100-$500/hr) vs traditional human/agency retainers.

**How to run:**
Open `index.html` in any browser. No build steps needed.
