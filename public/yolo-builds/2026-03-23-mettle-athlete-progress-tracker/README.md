# METTLE // Athlete Progress Tracker

Individual swimmer time progression analyzer with plateau detection, pattern recognition, and training recommendations.

## What It Does
Coaches paste time entries for any swimmer/event combo and get:
- Time progression chart with PB markers and trend line
- Plateau detection (stalled improvement window)
- Seasonal and meet-type performance patterns
- Training recommendations based on improvement rate
- Consistency scoring and improvement velocity analysis
- Export to text report, CSV, or JSON

## How to Run
Open `index.html` in any browser. Click "Load Sample Data" to see it in action.
