# METTLE // Season ROI Calculator

Interactive subscription economics modeler for METTLE swim coaching SaaS. Coaches and business analysts input team size, pricing tiers, churn rates, and cost structure to model 12-month financial projections.

## How to Run
Open `index.html` in any browser. No dependencies, no build step.

## Features
- **5-tab layout:** Inputs → Unit Economics → 12-Mo Projections → Scenarios → Upgrade Path
- **Unit economics:** ARPU, LTV, LTV:CAC ratio, payback period, break-even analysis with margin bars
- **12-month projections:** Canvas-rendered revenue/profit/cost curves with monthly breakdown table
- **Scenario analysis:** Bear/Base/Bull cases with churn sensitivity chart
- **Upgrade path modeling:** Tier migration simulation (Starter→Pro→Elite) with ARPU expansion tracking
- **Actionable insights:** Every panel generates context-aware recommendations based on the numbers
- **Export:** CSV and JSON for further analysis

## What's Missing / Next Steps
