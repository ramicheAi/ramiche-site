# Filament Dry Box Monitor — NOVA PrintLab

Interactive dashboard for tracking filament moisture levels, drying cycles, and material health across your spool inventory.

## Features

- **Live drying cycle tracker** with countdown timer, progress ring, and real-time humidity drop simulation
- **Spool inventory** with per-material moisture thresholds (PLA, PETG, ABS, TPU, Nylon)
- **Humidity trend chart** with 24h / 7d / 30d views
- **Alert system** — warning and critical states based on material-specific thresholds
- **Drying history log** with filterable entries (cycles, alerts, all)
- **Material reference table** — dry temps, times, and storage humidity targets
- **Add spool form** — register new spools into the inventory
- **Configurable alert thresholds** — warning/critical RH, auto-dry, polling interval

## Design

Single-page, linear layout. Dark theme. No dependencies — pure HTML/CSS/JS. Mobile responsive. Premium UI with animated progress indicators, live simulation, and interactive charts.
