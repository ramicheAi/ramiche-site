# NOVA Filament Inventory Manager

Track your filament spool stock, material properties, cost analysis, and print readiness in one clean interface.

## What It Does

Full spool inventory management for FDM 3D printing:
- **Spool tracking** — name, brand, material, color, weight remaining, price
- **Material intelligence** — auto-populated density, temp ranges, strengths/weaknesses per material type
- **Cost analysis** — price per gram, remaining value, total inventory value
- **Stock alerts** — visual indicators for low stock (<15%) and empty spools
- **Search & filter** — by name, brand, material type, or notes
- **CRUD operations** — add, edit, adjust weight, delete spools
- **CSV export** — full inventory export for spreadsheets
- **Persistent storage** — localStorage saves your data between sessions

## Supported Materials

PLA, PLA+, PETG, ABS, TPU, ASA, Silk PLA, CF-PLA, CF-PETG — each with preset properties.

## Features

- 12 pre-loaded seed spools (common brands/materials)
- Color-coded spool swatches with 20-color picker
- Expandable detail cards with print settings + material properties
- Stats dashboard: total spools, weight on hand, inventory value, low stock count
- Material filter pills for quick views
- Responsive layout (desktop → mobile)
- Dark theme, zero dependencies, single HTML file

## Target Users

FDM printer operators managing filament inventory across multiple materials and brands.
