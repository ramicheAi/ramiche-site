# METTLE // Pricing A/B Simulator

Sales intelligence tool for testing pricing psychology before going live. Compare 6 pricing strategies side-by-side with projected close rates, churn, LTV, ARR, and psychological trigger analysis.

## How to Run
Open `index.html` in any browser. No dependencies.

## Features
- **6 Pricing Strategies:** Anchor High, Per-Athlete, Value Stack, Flat Rate, Freemium + Upsell, Competitor Undercut
- **Side-by-side A/B comparison** with real-time metrics
- **Psychological trigger tags** (positive/negative/neutral per strategy)
- **12-month revenue projection** with ARR and LTV bar charts
- **Composite scoring** with auto-winner detection (40% ARR + 30% LTV + 20% close rate + 10% retention)
- **Objection handling scripts** that adapt per strategy — ready-to-use in sales calls
- **All 4 METTLE tiers** ($149/$349/$549/$899) with configurable team size
