# METTLE Practice Planner — Coach Workout Builder

Interactive swim practice set builder for coaches. Build structured workouts with drag-and-drop set groups, energy zone tracking, and exportable practice cards.

## How to Run

Open `index.html` in any browser. No build step needed.

## Features

- **Set Group Builder** — Warm-Up, Pre-Set, Main Set, Kick, Pull, Drill, Sprint, Cool-Down, Test Set groups
- **Quick Templates** — 8 pre-built set templates for fast practice construction
- **Energy Zone Tracking** — 7 zones (REC, EN1-EN3, SP1-SP3) with real-time distribution chart
- **Volume Calculator** — Auto-calculates total yards/meters, set count, estimated time
- **Drag & Drop Reorder** — Drag set groups to reorder the workout flow
- **Pool Course Support** — SCY (25y), SCM (25m), LCM (50m)
- **Export** — Text-formatted practice card with zone breakdown, copy to clipboard, or print
- **LocalStorage** — Save/load practices between sessions
- **Demo Mode** — Load a full 4,200-yard distance/threshold practice to see it in action
