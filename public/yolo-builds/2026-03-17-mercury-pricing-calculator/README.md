# Mercury Pricing Calculator

Value-based pricing engine for Ramiche Studio services. Enter service type and client budget, get a three-tier quote (Anchor/Target/Premium) with automatic floor pricing for 48h sprints.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- 3 service types: Brand Sprint, Content Retainer, Full Architecture
- Value-based pricing (10% of client value x complexity multiplier)
- Three-tier output: Anchor (75%), Target (100%), Premium (150%)
- 48h Sprint floor rate ($10,000 minimum)
- Scope descriptions per tier
- Print-ready quote output

## What's Missing

- PDF export
- CRM deal creation from quote
- Historical quote tracking
- Client-facing branded version
