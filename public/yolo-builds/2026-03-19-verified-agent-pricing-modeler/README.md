# Verified Agent Pricing Modeler

A pricing tool that calculates the maximum justifiable monthly agent retainer fee based on the client's current revenue, margins, and the agent's projected margin/revenue impact. 

## How to run it
Open `index.html` in any web browser. No build steps or servers required.

## What's missing / next steps
- Save/Export to PDF for client proposals
- Add industry benchmarks for default margin values

## A/B Comparison
Improves on the "Parallax Agent Arbitrage Calculator" by shifting the conversation from "how much money we save you vs human employees" to "how much value we create by expanding your margins", allowing for higher value-based pricing instead of cost-plus or time-and-materials.
