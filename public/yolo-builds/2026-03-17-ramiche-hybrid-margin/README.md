# Ramiche Studio Hybrid Margin Engine

An interactive financial modeling tool for Ramiche Studio to calculate gross margin expansion when substituting traditional human creative hours with high-leverage AI agent hours.

### How to Run
Open `index.html` in any modern web browser.

### Features
- Preset project structures (Brand Kit, Web Build, Social Retainer, Custom Agent)
- Slider controls for Human and Agent billable rates and hours
- Real-time KPIs for Client Price, Total Cost, Gross Profit, and Blended Margin
- Visualized value delivery mix and economic waterfall chart
- Dynamic text-based insight generation ("Simons Analysis")

### Next Steps / Missing
