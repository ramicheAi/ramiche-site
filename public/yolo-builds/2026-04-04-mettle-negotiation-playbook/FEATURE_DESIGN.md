# Feature Design: METTLE Negotiation Playbook

**Agent:** MERCURY
**Date:** 2026-04-04
**Status:** Complete

## Problem
Mercury has tools for proposals, objection handling, and pipeline tracking — but no structured negotiation framework. Sales conversations happen ad-hoc without a systematic strategy for concessions, walk-away points, or counter-offers.

## Solution
Interactive negotiation war room that takes prospect inputs and generates a complete negotiation playbook with:
- Leverage scoring (0-100) based on 8 factors
- Concession ladder with trade-for-trade economics
- BATNA analysis for both sides + ZOPA calculation
- Verbatim counter-offer scripts for 8 objection types
- 6-phase negotiation timeline
- 7 psychological tactics with deployment timing

## Key Decisions
- Concessions are trades, not gifts — every give has a get
- Walk-away points are concrete (floor price, # of free months, misalignment signals)
- Scripts use actual METTLE numbers (retention math, per-athlete pricing)
- Leverage score factors: timeline, current platform, objection count, team size, budget vs tier price

## Metrics
- Fills the negotiation gap in Mercury's sales tool suite
- Reuses proven financial models from earlier builds (churn reduction, LTV, per-athlete economics)
