# METTLE // Outbound Sequence Builder

Cold outreach campaign engine — enter target swim team info, select a psychological outreach strategy, get a personalized 5-touch email sequence with A/B subject lines, send timing, pipeline tracking, and multi-format export.

## How to Run

Open `index.html` in a browser. No build step, no dependencies.

## Features

- **6 outreach strategies:** Challenger, Proof Stack, Relationship Builder, Window Closer, Problem→Agitate→Solve, Warm Intro
- **5-touch personalized sequences** with dynamic merge variables (team name, athlete count, pain points, pricing, ROI math)
- **A/B subject line variants** for every touch
- **Send timing recommendations** with day-of-week optimization
- **Pipeline tracker** with stage progression (Cold → Warm → Hot → Won) and localStorage persistence
- **4 export formats:** Email copy, CSV pipeline, CRM JSON, send calendar
- **Pain point engine:** 6 pain categories with problem/agitate/solve copy frameworks
- **ROI math** auto-calculated per target team size and tier

## What's Missing / Next Steps

- Integration with actual email sending (SMTP/SendGrid)
- Open/reply tracking analytics
- Multi-sequence campaign management (run multiple targets in parallel)
- Connection to METTLE CRM for automatic lead creation
- Calendar integration for booking links in emails
