# METTLE Deal Objection Handler

Interactive sales objection classifier and response playbook engine for METTLE swim coaching SaaS. Paste a prospect's objection, get an instant response framework with empathy statements, reframes, proof points, METTLE-specific talk tracks, closing questions, and follow-up sequences.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- **8 objection categories:** Price, Timing, Competitor, Authority, Need, Trust, Complexity, Status Quo
- **NLP-style classifier** — weighted keyword matching with longer-keyword priority
- **Full response playbook per category:** reframe, empathy, proof point, talk track, closing question, 4-step follow-up
- **METTLE-specific talk tracks** — real responses using Saint Andrew's data, retention metrics, TeamUnify positioning
- **Quick-Fire Drill Mode** — random objections timed for speed practice with 4-criteria scorecard (empathy, reframe, proof, closing question)
- **Confidence rating** — 1-5 self-assessment per response to track weak areas
- **Objection history** with localStorage persistence
- **Category heatmap** — visual breakdown of which objections come up most + weakness analysis
- **14 example objections** and **20 drill scenarios** covering all 8 categories
- **Keyboard shortcuts** — Ctrl+Enter to submit
