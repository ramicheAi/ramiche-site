# Sales Velocity Dashboard

Interactive dashboard that calculates and visualizes **Sales Velocity** — the most important revenue metric: how much money flows through your pipeline per day.

## Formula
```
Velocity = (Opportunities × Win Rate × Avg Deal Size) ÷ Sales Cycle Length
```

## Features

- **Real-time velocity calculation** with 4 adjustable levers (opportunities, win rate, deal size, cycle length)
- **Lever Impact Analysis** — shows which lever gives the highest ROI if improved by 10%, with actionable recommendations
- **Scenario Snapshots** — save current settings, compare multiple scenarios side-by-side with % delta
- **Revenue Projections** — 30/90/180/365-day forecasts with cumulative sparkline chart
- **Sensitivity Matrix** — win rate × deal size grid showing monthly revenue at each intersection
- **Deal Economics** — deals/month, revenue per opportunity, total pipeline value
- **Presets** — SMB, Mid-Market, Enterprise, and METTLE SaaS configurations
- **Zero dependencies** — single HTML file, runs anywhere
