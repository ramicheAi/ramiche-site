# Material Stress Tester — NOVA

Interactive tool for predicting mechanical stress behavior across 6 FDM printing materials (PLA, PETG, ABS, ASA, TPU, Nylon) based on part geometry, load conditions, and environmental exposure.

## Features

- **Part Geometry Input:** Wall thickness, infill density, layer height, print orientation, part dimensions
- **Load Analysis:** 6 load types (tensile, compressive, bending, shear, impact, fatigue) with force and duration
- **Environmental Factors:** Temperature (-20°C to 120°C), UV, moisture, chemical, abrasion, food safety
- **Material Database:** Real-world mechanical properties for PLA, PETG, ABS, ASA, TPU, Nylon
- **Scored Rankings:** Each material scored 0-100 with visual bar chart comparison
- **Failure Warnings:** Context-aware warnings for delamination, heat softening, buckling, etc.
- **Print Settings Advice:** Adjusted slicer recommendations based on load analysis

## Usage

Open `index.html` in any browser. No dependencies, no build step. Single self-contained file.
