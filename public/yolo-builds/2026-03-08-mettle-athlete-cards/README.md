# METTLE Athlete Card Generator

Interactive athlete trading card generator for the METTLE platform. Coaches and parents can create shareable athlete profile cards with stats, level badges, event results, and custom themes — then download as PNG for social media sharing.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- **Live preview** — card updates in real-time as you type
- **6 METTLE levels** — Rookie through Legend, each with unique visual treatment (Legend gets golden sparkles)
- **Photo upload** — drop in an athlete headshot or use auto-generated initials
- **10 sports** — Swimming, Track & Field, Football, Basketball, Soccer, Baseball, Volleyball, Wrestling, Cross Country, Lacrosse
- **Event results** — add unlimited events with best times and placement medals (gold/silver/bronze)
- **6 color themes** — purple/red, blue/cyan, green/teal, red/gold, violet/pink, dark/gold
- **PNG export** — download a 760x1120 card image ready for Instagram/Twitter/print
- **4 demo presets** — Swimmer, Sprinter, Football, Legend
- **3D hover effect** — card tilts on mouse hover with holographic shine overlay
- **Saint Andrew's Aquatics** demo data — matches the METTLE beta team

## What's Missing / Next Steps

- Integration with METTLE's actual athlete data via Firebase
- Batch card generation for entire rosters
- Back-of-card design with season stats breakdown
- Animated card reveal (flip animation)
- Direct social sharing (share to Instagram Stories, Twitter)
- QR code linking back to athlete's METTLE profile
