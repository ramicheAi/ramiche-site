# METTLE Proposal Generator — MERCURY Sales & Revenue

Interactive proposal builder for METTLE swim team prospects. Input team details → get a polished, print-ready proposal with auto-tier recommendation, ROI projections, and competitive positioning.

## What It Does

- **Prospect Input** — Team name, contact, athlete count, groups, current platform, pain points.
- **Auto Tier Recommendation** — Recommends Starter ($149), Professional ($349), Program ($549), or Enterprise based on athlete count. Manual override available.
- **ROI Calculator** — Retention-based model: 20% industry churn baseline × METTLE engagement lift = athletes retained × avg monthly dues = revenue protected. Shows ROI multiple.
- **KPI Strip** — Monthly cost, per-athlete cost, retention lift %, ROI multiple at a glance.
- **Competitive Positioning** — Dynamic comparison vs TeamUnify/Swimmingly/spreadsheets. METTLE wins on gamification, Race AI, engagement.
- **Feature Breakdown** — Complete feature list by tier with inheritance.
- **Full Proposal Generation** — Branded, print-ready document with personalized messaging, ROI breakdown, next steps, and demo CTA.
- **Print/PDF** — CSS print styles for clean PDF export. Also supports HTML copy for email embedding.

## How to Run

Open `index.html` in any browser. No build step, no dependencies. Single HTML file.

## Sales Use Cases

1. **Cold outreach** — Generate proposal, attach as PDF to intro email
2. **Post-demo follow-up** — Customize with pain points discussed, send same day
3. **Budget justification** — ROI section helps coaches sell METTLE to their board
4. **Competitive displacement** — Platform-aware messaging for TeamUnify/Swimmingly shops
