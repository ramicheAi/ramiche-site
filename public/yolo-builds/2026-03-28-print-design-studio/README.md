# Print Design Studio

**Agent:** NOVA | **Date:** 2026-03-28 | **Lane:** Product

## What It Is

Customer-facing product configurator for Ramiche Studio's 3D print services. Non-CAD users design custom products through a guided 5-step flow and get an instant quote.

## Features

- **8 product templates:** Nameplate, Phone Case, Keychain, Desk Organizer, Coaster, Planter, Storage Box, Custom Shape
- **Live 3D preview** with CSS 3D transforms — drag to rotate, updates in real-time as dimensions/color change
- **6 materials** with per-gram pricing: PLA, PLA+, PETG, ABS, ASA, TPU 95A
- **12 filament colors** with visual swatches
- **Dimension configurator** with wall thickness and infill density sliders
- **Text engraving** with live preview on the 3D model
- **Real pricing engine:** shell volume + infill volume + material density + machine time + labor + 60% margin
- **Print time estimation** based on layer count and area
- **Quantity pricing** with per-unit breakdown
- **Spec summary** with all parameters in chip format
- **Order submission** with unique order ID generation

## Flow

1. Pick product template → auto-sets default dimensions
2. Adjust width/depth/height, wall thickness, infill
3. Choose material and color
4. Add optional engraving text + special instructions
5. Review 3D preview, quote, specs → submit

## Why This Matters

Most 3D print customers don't have CAD files. This bridges the gap between "I want a custom thing" and "here's a printable design with a price." Revenue tool that converts browsers into buyers.
