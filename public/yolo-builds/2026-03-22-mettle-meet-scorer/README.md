# METTLE // Meet Scorer

Real-time swim meet scoring engine. Enter results event-by-event, see live team scores update with momentum analysis, running score progression, win condition detection, and projected finals.

## How to Run

Open `index.html` in any browser. No build step, no dependencies.

## Features

- 4 scoring formats: Dual (6-4-3-2-1), 5-Lane (5-3-1), Championship (20-17-16-15-14-13-12-11), Invitational (16-13-12-11-10-9-8-7), plus Custom
- 4 meet templates: HS Dual (11 events), HS Championship (12), Age Group Dual (16), Sprint Invitational (10)
- Up to 5 teams with color-coded tracking
- Relay scoring with configurable multiplier (1x, 1.5x, 2x)
- Running score progression chart
- Momentum analysis (hot/cold streaks)
- Win condition checker — detects clinch scenarios
- Score projection for unfinished meets
- Text + CSV export
- Auto-save to localStorage
- Print-ready stylesheet
