# FD: Sales Battlecard Generator

**Status:** Complete
**Owner:** MERCURY
**Created:** 2026-04-12 02:00
**Lane:** Sales & Revenue

## Objective
Interactive competitive intelligence battlecard tool with pre-loaded METTLE competitors, feature matrices, objection handles, talking points, landmines, and export capabilities.

## Acceptance Criteria
- [x] 4 pre-loaded competitors with full battlecard data (TeamUnify, SwimTopia, Deck Pass, OnDeck)
- [x] 14-feature comparison matrix with advantage indicators
- [x] Objection handles with question/response pairs per competitor
- [x] Talking points and landmines sections
- [x] Win/loss record with visual bar and percentage
- [x] Pricing comparison cards (METTLE vs competitor)
- [x] Markdown export (single card + all cards)
- [x] Add new competitor form
- [x] Summary bar with aggregate competitive metrics
- [x] Threat level classification (high/medium/low)

## A/B Context
Previous builds this relates to: mettle-lead-qualifier (Apr 5), mettle-negotiation-playbook (Apr 4)
Variable changed: Moved from deal-stage tools to pre-deal competitive intelligence
Expected improvement: Reps enter every competitive deal pre-armed with exact objection handles and competitor weaknesses

## What Worked / What Didn't
The objection handle format (prospect quote → Mercury response) is immediately usable in live calls. Landmines section is the highest-value add — knowing what NOT to say prevents more lost deals than knowing what to say. Feature matrix with advantage delta column makes it scannable. Pre-loading 4 real competitors with real sales data makes this useful on day one, not after hours of setup. Export to markdown means reps can paste directly into CRM notes before a call.
