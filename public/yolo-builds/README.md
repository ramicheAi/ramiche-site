# YOLO Overnight Builds

Each night, the squad generates one wild project idea and builds a working prototype.

## Structure
- `dashboard.html` — running feed of all builds (newest first, filterable)
- `builds.json` — structured log of all builds
- `YYYY-MM-DD-project-name/` — individual project folders

## Rules
1. Generate the idea from recent context, memory, and interests
2. Build the simplest version that proves it works
3. Default stack: Python backend, React/HTML for UI
4. If stuck 15 min, pivot approach
5. Bias toward "shipped and ugly" over "planned and pretty"
