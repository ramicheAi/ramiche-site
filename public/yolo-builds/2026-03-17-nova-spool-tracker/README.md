# Nova SpoolTracker

A lightweight frontend for tracking 3D printer filament spools. It uses localStorage to maintain state and allows the user to add, update, and remove spools with visual indicators for remaining material.

## How to Run
Open `index.html` in your web browser.

## Missing/Next Steps
- Integration with Bambu Lab API to auto-deduct filament when a print starts
- Support for spool weighting instead of manual gram deduction
- Filtering and search functionalities for large inventory
