# Verified Agent Business — Enterprise ROI Calculator

Interactive ROI calculator that lets enterprise prospects input their current staffing costs and see projected savings from deploying certified AI agents across 5 verticals.

## How to Run

Open `index.html` in any browser. No dependencies, no build step.

## Features

- **5 Verticals:** Sales ($150/hr), Security ($250/hr), Legal ($350/hr), Finance ($200/hr), Psychology ($175/hr)
- **Dynamic Inputs:** Employee count, salary, benefits overhead, training costs, automatable hours
- **Agent Model Controls:** Automation rate slider, speed multiplier, monthly hours, implementation timeline
- **Results Dashboard:** Annual savings, ROI %, hours reclaimed, payback period
- **12-Month Projection Chart:** Canvas-rendered cumulative savings vs. cost with break-even marker
- **Cost Breakdown Table:** Side-by-side current vs. with-agents comparison
- **Vertical-Specific Insights:** Capabilities and industry benchmarks per agent type
- **Ambient UI:** Particles, glow effects, gradient accents — Parallax design language
