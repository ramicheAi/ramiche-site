# Upwork Profile — Parallax

## Profile Title
AI Operations Architect | Deploy Autonomous Agent Workforces for Your Business

## Overview (2000 char max)
We build and deploy autonomous AI agent teams that run your business operations 24/7.

Not chatbots. Not single-prompt automations. Full operating systems — persistent agents with identity, memory, scheduled tasks, and the ability to operate your existing tools.

What we deliver:

→ Custom AI agent teams (operations, sales, support, content, security)
→ Automated workflow orchestration across your entire business
→ Browser automation agents that operate legacy portals and CRMs
→ OpenClaw platform setup and configuration
→ Claude/GPT skill development and prompt engineering

Our production system runs 19 specialized agents handling operations, sales, customer support, content creation, cybersecurity, and financial analytics — autonomously. We've packaged this architecture so any business can deploy the same thing.

Key differentiator: Every agent has a "SOUL file" — a full identity document with experiential beliefs, anti-patterns, and domain expertise. This produces dramatically better results than raw prompts on any model.

Results we've delivered:
• Deployed 4-agent workforce in under 30 minutes (verified end-to-end)
• Built automated email delivery pipelines with zero manual intervention
• Created multi-model orchestration systems (Claude, DeepSeek, Kimi, GLM)
• Developed browser automation agents for legacy web portal operations

Tech: OpenClaw, Claude API, OpenAI, Next.js, Stripe, Vercel, Node.js, Python, Firebase
