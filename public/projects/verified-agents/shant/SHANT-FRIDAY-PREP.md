## FRIDAY SESSION FLOW (2.5 hours)

### Phase 1: UNDERSTAND (15 min)
- "Walk me through your busiest day this week"
- Listen. Mirror back. Show we understand his workflow.
- Identify the 3 biggest time sinks.

### Phase 2: WOW (15 min)
- Open Command Center on screen share — his first look at a live 19-agent system
- Show one agent responding to a task in real time
- "This runs 24/7 while you sleep. Every morning, there's new work waiting for you."

### Phase 3: SECURITY (10 min)
- Address head-on: "Someone told you there are security risks. Let's talk about that."
- Show: local-only execution, gateway protection, ClawGuard scan
- Explain: "Nothing leaves your machine unless you tell it to"
- One-pager handoff: what's locked, what's controlled, what's monitored

### Phase 4: SETUP (60 min)
- Mac Mini unboxing → install script → OpenClaw running
- Pre-configured agents for his use case:
  - **COPY** — writes funnel copy, email sequences, ad text
  - **DESIGN** — generates layouts, mockups, brand assets
  - **BUILDER** — creates pages, basic GHL integrations
  - **ANALYST** — tracks campaign performance, suggests optimizations
  - **SCHEDULER** — content calendar, posting pipeline
- Walk through each agent's role
- Set up his Telegram/WhatsApp channel connection

### Phase 5: FIRST WIN (30 min)
- Give the system a REAL task from his queue
- Watch it execute: research → copy → draft → output
- He sees his own work getting done without him
- "You just saved 3 hours. This happens every day now."

### Phase 6: YAUGGY + MUSIC (15 min)
- Brief him on Yauggy: Christian R&B, signed to Parallax/Baba Studio
- Hand over EPK + streaming links + visual assets
- Discuss funnel strategy: Spotify playlist → IG → YouTube → streaming growth
- Define: what he'll build, timeline, success metrics

### Phase 7: CLOSE (15 min)
- Define trade deliverables + deadlines:
  - His: Parallax funnel wireframe by [date], Yauggy campaign brief by [date]
  - Ours: Ongoing agent support, design deliverables, weekly check-in
- Set next meeting (1 week out)
- Permission to record session for course content
- "If you know anyone else who needs this, we're launching a consulting program"
