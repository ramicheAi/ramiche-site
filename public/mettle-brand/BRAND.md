# METTLE Brand Guidelines

## Brand Name
**METTLE** — always uppercase in wordmark/logo. Can be title case (Mettle) in body text.

## Tagline
"Where athletes find their edge."

## Colors

### Primary
- **Cyan** `#00F0FF` — primary brand color, used for logo, UI accents, interactive elements
- **Deep Cyan** `#0088AA` — gradient endpoint, used for depth

### Accent
- **Flame Orange** `#FF6B35` — energy, determination, fire. Used sparingly for highlights and CTAs
- **Warm Orange** `#FFB347` — gradient endpoint for flame motif

### Neutrals
- **Background Dark** `#0A0E17` — primary dark background
- **Surface** `#111827` — card/panel backgrounds
- **Border** `#1E293B` — subtle borders
- **Text Primary** `#F1F5F9` — main text on dark
- **Text Secondary** `#94A3B8` — secondary/muted text

## Logo Variants

| File | Use Case |
|------|----------|
| `mettle-logo-full.svg` | Primary — icon + wordmark. Use when space allows |
| `mettle-wordmark.svg` | Text only. Use when icon context is already established |
| `mettle-icon.svg` | Icon only. App icons, favicons, avatars, small spaces |
| `mettle-icon-white.svg` | White variant for dark/photo backgrounds |

## Icon Anatomy
- **Hexagon** — strength, structure, molecular resilience
- **M zigzag** — mountain range / heartbeat / perseverance through peaks and valleys
- **Flame crown** — the fire within, determination, the "mettle" inside every athlete

## Typography
- **Headlines:** Bold, uppercase, tight tracking (matches wordmark energy)
- **Body:** Clean sans-serif (Inter, system-ui)
- **Monospace:** JetBrains Mono (data/stats displays)

## Usage Rules
1. Minimum clear space around logo: 1x the height of the hexagon icon
2. Minimum icon size: 24px
3. Never stretch, rotate, or add effects to the logo
4. On light backgrounds, use the cyan version. On dark, use cyan or white
5. The flame accent is ALWAYS orange — never recolor it
6. Do not place the logo on busy/complex backgrounds without a container

## Voice & Tone
- Confident, not arrogant
- Athletic, not bro-culture
- Technical, not corporate
- Playful, not childish
- Inclusive across all sports and skill levels
